# Inflammatory Messages Experiment (Paper Section 7.3)

Information propagation and emotional dynamics under different moderation strategies.

## Paper Findings
- Inflammatory messages have stronger viral potential than regular content
- Node intervention (account ban) > Edge intervention (connection removal)
- Sharing driven by emotional reactions and social responsibility

## Run

```bash
pip install agentsociety2 python-dotenv
python run_inflammatory.py
```

Results saved to `results/inflammatory/results.json`.
