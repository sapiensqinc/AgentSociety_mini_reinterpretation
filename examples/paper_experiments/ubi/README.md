# Universal Basic Income Experiment (Paper Section 7.4)

Compares economic and social metrics with/without monthly UBI payment.

## Paper Findings
- UBI ($1000/month) increases consumption levels
- UBI reduces depression levels (measured by CES-D scale)
- Agent interviews: concerns about interest rates, long-term benefits, savings

## Run

```bash
pip install agentsociety2 python-dotenv
python run_ubi.py
```

Results saved to `results/ubi/results.json`.
